Kerala Salary Prediction System |

Using a real-world dataset, I developed a complete ML pipeline including data preprocessing, feature encoding, model training with Random Forest Regressor, and performance evaluation using R² and MAE metrics. The system predicts candidate salary based on experience, education, role, company type, and location, and includes an interactive Streamlit deployment with salary trend visualization. This project demonstrates my practical understanding of regression modeling, feature importance analysis, and deploying ML solutions for real-time HR decision support.
